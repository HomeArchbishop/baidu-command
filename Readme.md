# baidu-command

- [简体中文](#zh-cn)

A command to quick open your browser and search.

### Installation and Usage 安装及使用
```bash
# installation
npm i -g baidu-command

# usage 使用
# Tip: there is no need to quote, I'll process all.
baidu something to search
```
your browser is opening, enjoy.
